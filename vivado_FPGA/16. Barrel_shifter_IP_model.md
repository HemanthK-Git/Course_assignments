BARREL SHIFTER: 

1) mux_4x1 code:
/*********************************************
    `timescale 1ns/1ps
    
    module mux_4x1(
        input i0,i1,i2,i3,sel0,sel1,
        output reg y
        );
        
        always @(*)    
            begin
            
                if (~sel1 & ~sel0)
                    y=i0;
                else if (~sel1 & sel0)
                         y=i1; 
                else if (sel1 & ~sel0)
                         y=i2;
                else if (sel1 & sel0)
                      y=i3;
         
            end
    endmodule
*********************************************************/
